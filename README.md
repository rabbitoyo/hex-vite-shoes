### 🛠 使用技術

- SCSS (變數、mixin、迴圈)
- Vite (ejs 模板管理)

### 💻 安裝執行

- 將專案 clone 到本地端

```bash
git clone https://github.com/rabbitoyo/hex-vite-shoes.git
```

- 安裝執行

```bash
npm install				// 安裝
npm run dev				// 執行
```

- 瀏覽器訪問

```bash
http://localhost:5173/
```
